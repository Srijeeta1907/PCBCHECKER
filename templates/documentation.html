System Documentation

Guide for operating the AI-powered PCB inspection system.

1. Overview

QualityCheck utilizes a YOLO (You Only Look Once) deep learning architecture to identify defects in real-time. The system is divided into two specialized "brains": one for Bare Boards (detecting copper-level flaws) and one for Assembled Boards (detecting component placement errors).

2. How to Use

  • Upload Image: Select a high-resolution top-down photo of the PCB.
  • Select Model: Choose "Bare Board" for unpopulated PCBs or "Assembled" for finished products.
  • Analyze: The AI will return a list of detections with a confidence score and location boxes.

3. Defect Types

Bare Board Faults

Detects: Short circuits, Open circuits, Spurious copper, and Mouse-bites.

Assembled Faults

Detects: Missing components, Misaligned parts, and Solder bridges.

4. Reading the Analysis Documents

After the AI processes a PCB image, it generates an inspection document. Here is how to read and interpret the results:

Bounding Boxes

The visual document displays colored boxes around detected faults. Each box is labeled with the specific defect type (e.g., "Short", "Missing Component"). This allows you to instantly pinpoint the exact physical location of the error on the board.

Confidence Scores

Next to each bounding box label is a percentage (e.g., 98%). This represents the AI model's certainty that the detection is accurate. Any score above your set threshold is flagged for review.

Summary Reports

The system compiles a structured text document summarizing the total count of each defect type, the processing time, and a timestamp. This gives a quick, high-level overview of the board's overall health without needing to check every single visual box.

Expert Tip

For best results, ensure the lighting is even. Harsh shadows on a bare board can sometimes be misinterpreted by the AI as a trace defect. Use a white background for bare board scans.
